The
application
uses
environment
variables
.
An
environment
variable
exposed
to
the
user
is
vulnerable
to
a
buffer
overflow
.
The
vulnerable
environment
variable
uses
untrusted
data
.
Tainted
data
used
in
the
environment
variables
is
not
properly
validated
.
For
instance
boundary
checking
is
not
done
before
copying
the
input
data
to
a
buffer
.
